# Leibniz
The compiler and interpreter for the Leibniz language, along with various derivative works.   Leibniz语言的编译器和解释器以及各种衍生产物.
以下为作者要做的事情和已经做完的事情.此项目开发团队目前只有作者,欲加入请联系作者QQ3187909557.感谢支持作者的人.
